## קימפול (Windows)
התקן MinGW או MSYS2 והרץ את הפקודה הזו:
```
g++ main.cpp -o "אישור סיומות" -mwindows
```
כעת אמור להיווצר קובץ הרצה בשם "אישור סיומות.exe".

כדי שהתוכנה תעלה באתחול, אפשר להעביר אותה לתיקיית `shell:startup`
